#+title: Logic synthesis

The process os logic synthesis consists in: given a logic function with a desired output, simplify the function so it can be implemented in a circuit with the desired behavior and a minimal cost.

The are serveral ways to realize the logic synthesis, in the book we are presented to two ways:

 1. Obtaining the /canonical expression/ using [[id:e23c5dc3-934a-4f55-b401-be380770c9f3][minterms or maxterms]] and then minimizing the expression using [[id:1a9b753e-e05d-4dba-85a0-b18d6583a7d3][Boolean algebra]].
 2. Obtaining the minimal-cost expression using [[id:0d5b2d2b-f9a8-478f-911a-f6ab160833a9][Karnaugh maps]].
