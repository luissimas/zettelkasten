#+title: Bits de uma arquitetura

* Bits de uma arquitetura
Não há uma definição clara do que a quantidade de bits no nome da arquitetura significa (32 ou 64bits), visto que isso pode descrever diversas partes da arquitetura como: - Descrever a quantidade de bits usadas para representar inteiros endereços de memória. - Descrever a quantidade de bits dos registradores de uma CPU (ou ALU). - Descrever a quantidade de bits do barramento de dados do computador.
